WORLDVIEW / DREAM / FATE LINE

一个诞生于宇宙撕裂之后、在无序中求生的世界。梦是它降下的锚点，命运线是权柄的形状，十二司游则行走在灵层与初始宇宙的边界上。

理解灵层
底层原理

量子引力修正与反弹膨胀

灵层并非普通异世界。它是母宇宙撕裂后分离出的残缺宇宙，继承了更混乱、更高熵的量子涨落。

梦境锚点

梦让两界规则重叠

司游通过梦在初始宇宙设立锚点，让两个世界的规则开始重叠。梦境越深入，边界越难回收。

命运线

无法跳出的最高位概念

命运先于一切存在。命运线是命运赠与个体的具体化权柄，也是异能、权柄与反叛的源头。

SCRIPT SYSTEM

环文与世界观文字

环文不是普通输入法，而是一套概念文字：用环数、点位、刻痕与中心标记，把世界观概念变成可读的符号结构。

基础构成

  • 环数：概念的层级与类别
  • 点位：语义方向与坐标
  • 刻痕：竖、横、斜、点、断、双
  • 中心：空、点、圆、实、裂、锁、双、旋

原未央

起源、未完成与中心显现。相似的外部结构，通过中心状态区分意义。
